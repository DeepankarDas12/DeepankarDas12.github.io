Latex beamer in SVG format

Advantages over PDF beamer

  • Videos and animations can be presented without depending on flash (which is no longer supported in most web browsers).
  • Custom pointers can be used independent of system OS.
  • Raster images are linked instead of being embedded, therefore can be changed without recompiling.

Disadvantages over PDF beamer

  • Does not produce a single file, therefore directory structure must be preserved.
  • Needs extra work to get a printout.
  • Hypertex is used instead of hyperref, so some functionalities may be limited.

How to do it?

I have done it in MiKTeX but other LaTeX distributions should also work. The required files are here.

The above link contains a zip file. The main components that you need are 'SVGPPT.sty' and 'pointer' folder. The 'pointer' folder contains some custom pointer icons. You can also add your own icons to it. The file 'SVGPPT.sty' is the package that is to be used in your tex files. Both of these should be in the folder containing your tex file.

The zip file also contains an "Example.tex" as a sample document. It should be compiled using the following commands (open terminal in the 'Example' folder and proceed),

  • dvilualatex -synctex=1 -interaction=nonstopmode "Example"
  • dvisvgm --linkmark=none --bbox=papersize --no-fonts --zoom=-1 --page=- "Example"

You might have to run the above commands twice to ensure that the hypertex links work. You can view the presentation by opening the 'Example.html' You can change slides with arrow keys. The compiled beamer should like the following.

A TeXstudio profile is provided here which can be loaded into TeXstudio to automate the above steps. Similar profiles can also be created with other editors.
